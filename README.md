# Персональний помічник

Командний проект з курсу **Python Foundation (Neoversity)**.

Консольний додаток для управління контактами та нотатками з підтримкою збереження даних у CSV-файли, інтерактивною навігацією та таблицями.

---

## Структура проекту

```
personal_assistant/
├── main.py           # Точка входу, CLI меню
├── contacts.py       # Модуль контактів (Contact, ContactManager, Validator)
├── notes.py          # Модуль нотаток (Note, NoteManager)
├── storage.py        # Універсальний клас збереження даних у CSV
├── requirements.txt  # Залежності
├── README.md         # Документація
└── data/
    ├── contacts.csv  # Зберігається автоматично
    └── notes.csv     # Зберігається автоматично
```

---

## Вимоги

- Python **3.10+**
- Залежності з `requirements.txt`

## Функціонал

### Контакти
- Додавання контакту (ім'я, адреса, телефон, email, день народження)
- Пошук за іменем, телефоном або email
- Редагування з підтримкою дефолтних значень (Enter — залишити поточне)
- Видалення з вибором із дублів через стрілкове меню
- Перегляд усіх контактів у вигляді таблиці
- Іменинники — контакти з днем народження протягом N днів
- Валідація телефону, email та дати народження при введенні

### Нотатки
- Додавання нотатки (заголовок, текст, теги через кому)
- Пошук за заголовком, текстом або тегами
- Редагування з підтримкою дефолтних значень (Enter — залишити поточне)
- Видалення з вибором із дублів через стрілкове меню
- Перегляд усіх нотаток у вигляді таблиці з кольоровими тегами
- Пошук за тегом *(бонусне завдання)*
- Допускаються нотатки з однаковими заголовками

### Збереження даних
- Усі дані зберігаються у CSV-файли у папці `data/`
- Дані не втрачаються після перезапуску програми

| Бібліотека | Призначення |
|-----------|-------------|
| `questionary` | Інтерактивні меню зі стрілковою навігацією |

| `re` | Валідація телефону та email |
| `datetime` | Робота з датами народження |

# Персональний помічник

Командний проект з курсу **Python Foundation (Neoversity)**.

Консольний додаток для управління контактами та нотатками з підтримкою збереження даних у CSV-файли.

---

## Структура проекту

```
personal_assistant/
├── main.py           # Точка входу, CLI меню
├── contacts.py       # Модуль контактів (Contact, ContactManager, Validator)
├── notes.py          # Модуль нотаток (Note, NoteManager)
├── storage.py        # Універсальний клас збереження даних у CSV
├── requirements.txt  # Залежності
├── README.md         # Документація
└── data/
    ├── contacts.csv  # Зберігається автоматично
    └── notes.csv     # Зберігається автоматично
```

---

## Вимоги

- Python **3.10+**
- Залежності з `requirements.txt`

---

## Встановлення та запуск

```bash
# 1. Клонувати або завантажити проект

# 2. Встановити залежності
pip install -r requirements.txt

# 3. Запустити
python main.py
```

## Функціонал

### Контакти
- Додавання контакту (ім'я, адреса, телефон, email, день народження)
- Пошук за іменем, телефоном або email
- Редагування контакту
- Видалення контакту
- Перегляд усіх контактів
- Виведення іменинників — контактів, у яких день народження протягом N днів
- Валідація телефону та email при введенні

### Нотатки
- Додавання нотатки (заголовок, текст, теги)
- Пошук за заголовком, текстом або тегами
- Редагування нотатки
- Видалення нотатки
- Перегляд усіх нотаток
- Пошук за тегом *(бонусне завдання)*

### Збереження даних
- Усі дані зберігаються у CSV-файли у папці `data/`
- Дані не втрачаються після перезапуску програми
